Markdown的语法

==注:== 键入字符后，要紧跟着一个空格才能被识别为Markdown语法。否则只会被当作普通字符。

一、标题
  • 使用 # 可以表示标题
  • 其中 # 号的个数可以决定标题的级别
    • 个数越少，标题越高级 【如：一级标题只有一个 # 号】
    • 最多只能6个，因为标题最多只有六级标题

二、字体

2.1 斜体

  • 分别用一个*号将文字两端包起来

  • 也可以用一个下划线 _ 将文字两端括起来

    文字

    文字

    文字

2.6 高亮
  • 可以用两个 = 将要高亮的文本括起来

三、引用

  • 使用 > 可以表示引用
  • 同样是通过个数来表示层级

四、分割线

  • 使用三个 + 号来表示分割线
  • 也可以使用三个 - 号来表示
  • 也可以使用
五、表
5.1 列表

  • 无序列表使用 - + * 任何一种都可以

  • 有序列表使用数字加点 . 来表示

5.2 表

我们可以设置对齐方式：

  • :-表示左对齐
  • -:表示右对齐
  • :-:表示中间对齐
六、代码

  • 单行代码：使用 ` 将代码两端括起来即可

    create database hero;

  • 代码块：使用三个 ``` 后，再输入语言后键入回车键即可生成代码块
